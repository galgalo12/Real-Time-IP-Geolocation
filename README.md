# 🔍 Real-Time IP Geolocation Tool (Python)

This project is a Python-based tool that retrieves and displays real-time IP geolocation data using the ipinfo.io API. It allows users to input any IP address (or use their own) and returns detailed location and network information in a structured JSON format.

---

## 🚀 Features
- 🌐 Fetches geolocation data for any public IP address  
- 📍 Displays country, region, city, ISP, and coordinates  
- 🧾 Outputs results in clean, formatted JSON  
- ⚡ Handles API errors, invalid inputs, and network issues gracefully  
- 🔁 Interactive CLI menu for continuous use  

---

## 🛠️ Technologies Used
- Python  
- requests library (API calls)  
- json (output formatting)
